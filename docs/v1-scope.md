\# Hurricane Creek App – Version 1 Scope



\## Project Overview

This project is a mobile application intended for use at the Hurricane Creek Field Research Site.

The goal of Version 1 is to allow users to capture and submit georeferenced images securely for later administrative review.



---



\## In Scope (Version 1)

\- Mobile application built with React Native (Expo)

\- User authentication (admin-managed accounts)

\- Photo capture within the app

\- Optional GPS metadata with explicit user consent

\- Manual zone/area selection if GPS is unavailable or disabled

\- Image resizing/compression prior to upload

\- Secure backend API

\- Custom-designed database schema

\- Administrator-only interface for reviewing and moderating submissions



---



\## Future Scope (Version 1.5 / Version 2)

\- Interactive map with live pin placement

\- Guided tours and educational content

\- Public viewing of images

\- Safety features (location sharing, emergency guidance)

\- Trusted user roles

\- Advanced filtering and search
